UI/UX Reference Breakdown — Bongo-Fi Finance Dashboard

This layout is a premium fintech dashboard with a soft neumorphic / glassmorphism-inspired interface. It uses a light grey canvas, rounded cards, minimal iconography, black typography, lime green accents, and modular dashboard widgets.

The design should be reproduced as a desktop-first financial analytics/admin dashboard, with a responsive mobile version that stacks cards vertically.

1. Project Manager Specification
Product Type

Fintech / finance tracking dashboard for monitoring:

Account balance
Wallets
Cards
Income
Spending
Monthly limits
Recent transaction activity
Financial performance analytics
Main Page Purpose

The screen is an Overview dashboard. It gives the user an instant snapshot of financial health, available balance, income, spending, cards, and recent activity.

Core User Goals

The user should be able to:

View their total account balance.
Transfer or request funds quickly.
See wallet balances by currency.
Track monthly spending limits.
View linked cards.
Monitor income, spending, revenue, and earnings.
Analyse profit/loss through a chart.
Review recent transactions.
Search and filter transaction history.
Navigate between dashboard sections.

17. Graphic Designer Notes
Visual Direction

This dashboard has a premium SaaS finance aesthetic. The design relies on:

Smooth rounded corners
Light surfaces
Dark green contrast
Lime green performance accents
Clean iconography
Minimal visual noise
Soft shadows
Spacious card rhythm
Key Design Characteristics
Rounded everything
Pills, cards, buttons, chart bars, table rows.
Soft shadows
No harsh outlines.
Cards should feel elevated but not floating too much.
Lime green as action color
Used sparingly for positive growth, chart profit, and Transfer button.
Black used for authority
Active navigation, active sidebar icon, chart loss bars, checkbox.
Micro icons
Icons should remain small and subtle.
Dashboard should breathe
Avoid overcrowding even though there is a lot of information.

22. Exact Build Prompt for Developer

Use this prompt to brief a developer or AI builder:

Build a premium fintech dashboard UI inspired by a soft neumorphic finance admin interface. The screen should sit inside a large rounded white/off-white dashboard shell on a light grey background.

Create a desktop dashboard with a top navigation bar, left icon sidebar, greeting section, balance card, 2x2 KPI metric cards, total income bar chart, monthly spending limit progress card, my cards widget, and recent activities table.

Use a clean Inter-style font, off-white cards, soft shadows, rounded corners, black primary text, muted grey subtext, lime green accent buttons, dark green gradients, and black active states.

The top navigation should include the Bongo-Fi logo, tabs for Overview, Activity, Manage, Program, Account, Reports, search/notification icons, and a user profile pill.

The sidebar should be icon-only with a black active dashboard icon and subtle grey inactive icons.

The balance card should show total balance, currency dropdown, growth indicator, Transfer and Request pill buttons, and three wallet mini-cards for USD, EUR, and GBP.

The KPI area should have four cards: Total Earnings, Total Spending, Total Income, and Total Revenue. The Total Earnings card should be highlighted with a dark green/lime gradient.

The chart card should show Total Income with a profit/loss stacked bar chart from January to August using lime green for profit and black for loss.

The lower section should include a Monthly Spending Limit progress bar, two dark credit card previews, and a Recent Activities table with search, filter, checkboxes, order ID, activity, price, status, date, and row action menu.

Add subtle animations: dashboard fade-in, cards stagger upward, chart bars grow from bottom, table rows hover softly, buttons scale slightly on hover, and the spending bar animates from zero.

The result should feel like a polished 2026 SaaS fintech dashboard: minimal, premium, rounded, soft, data-rich, and visually calm.
23. Critical Reproduction Notes

To reproduce this reference accurately, do not make it overly colorful. The entire quality of the design comes from restraint.

The key details to preserve are:

Light grey outer canvas
Large rounded dashboard shell
Soft white cards
Black active navigation pills
Lime green financial accents
Dark green gradient KPI card
Icon-only left sidebar
Rounded chart bars
Minimal transaction table
Premium spacing and subtle shadows
