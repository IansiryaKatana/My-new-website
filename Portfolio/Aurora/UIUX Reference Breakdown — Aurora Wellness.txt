UI/UX Reference Breakdown — Wellness / Self-Care Landing Page

This concept is a premium wellness, nutrition, and guided self-care landing page. The design language is soft, editorial, calm, clinical-but-human, and image-led. It combines lifestyle photography, minimal typography, rounded cards, and subtle product/service commerce UI.

The page should feel like a mix of:

Wellness platform
Nutrition consultation service
Personal health subscription
Premium self-care brand
Light digital therapy / coaching product
1. Project Manager View
Overall Page Objective

The page is designed to sell a calm, personalized wellness experience. It introduces the brand, builds trust through lifestyle imagery and reviews, presents treatment/nutrition solutions, showcases product formulas, and closes with testimonials, FAQs, and footer conversion.

The experience should not feel like aggressive sales. It should feel guided, intimate, natural, and expert-led.

2. UI/UX Engineer View
Layout System

The page uses a 12-column desktop grid with generous whitespace. Most sections appear constrained inside a large max-width container, while the hero and testimonial section use a fuller width.

Recommended container system:

.page-container {
  max-width: 1440px;
  margin: 0 auto;
  padding: 0 40px;
}

.section {
  padding: 80px 0;
}

Mobile should collapse into a single-column layout with large image cards stacked vertically.

Visual Rhythm

The layout alternates between:

Full-width cinematic hero
White editorial sections
Soft gray product/treatment blocks
Dark green trust section
Clean FAQ
Black footer

This creates a calm scroll journey rather than a repetitive page.

3. Motion Graphics Developer View
Overall Motion Language

Motion should be slow, soft, and premium.

Avoid:

Fast bouncy effects
Harsh zooms
Overly playful animation
Heavy 3D transitions

Use:

Smooth fade-ins
Gentle image scale
Mask reveals
Text line reveals
Slow parallax
Soft hover transitions
Hero Motion
On Page Load
Background image/video fades in.
Background slowly scales from 1.08 to 1.0.
Navigation fades down.
Headline appears line by line.
CTA fades in after headline.

Recommended timing:

heroImage: 1200ms ease-out
nav: 600ms delay 200ms
headline: 900ms delay 400ms
cta: 500ms delay 900ms
Background Treatment

Hero image should have a subtle breathing motion:

transform: scale(1.05);
animation: slowZoom 14s ease-out forwards;
Scroll Animation

Each section should use scroll-triggered reveals.

Suggested Pattern
Section label: fade up first
Heading: line mask reveal
Images/cards: fade up with slight scale
Buttons: fade in last

Example:

Section enters viewport:
0ms label
150ms heading
300ms image/card group
500ms supporting text
650ms CTA
Card Hover Effects

Cards should have minimal movement.

.card:hover {
  transform: translateY(-4px);
}

Image hover:

.image-card img:hover {
  transform: scale(1.04);
}

Use a long transition:

transition: all 700ms cubic-bezier(.16, 1, .3, 1);
Treatment List Motion

When selecting a treatment:

Active item expands slightly
Description fades in
CTA slides/fades in
Image card changes with crossfade
Step marker fills subtly

Do not use hard tab switching. It should feel fluid.

Product Section Motion

Product carousel behavior:

Selected product image slides softly from right to left
Main product card fades and scales
Thumbnail opacity changes
CTA pill gently appears
Testimonial Motion

Carousel should use:

Fade transition
Slight vertical movement
Portrait crossfade
Quote text line reveal

Controls should have hover opacity and soft background fill.

Performance Requirements

Because this page is image-heavy, development must prioritize performance.

Required Optimizations
Use responsive images.
Use WebP/AVIF formats.
Lazy-load below-the-fold images.
Preload hero image/video.
Compress video background heavily.
Avoid loading all carousel images upfront.
Use skeleton placeholders if content is CMS-driven.

Example:

<Image
  src={image}
  alt={alt}
  fill
  priority={isHero}
  sizes="(max-width: 768px) 100vw, 50vw"
/>

7. Prompt for Rebuilding This UI

Use this as the production direction:

Create a premium wellness and self-care landing page with a soft editorial design system. The page should use cinematic lifestyle photography, warm off-white backgrounds, deep forest green accents, pale lime highlight cards, minimal pill buttons, and large light-weight sans-serif typography. The layout should feel calm, natural, clinical, and human. Build sections for a full-screen hero, mindful self-help philosophy, nutrition support, treatment discovery, product formula showcase, testimonials, FAQs, and footer. Use generous spacing, rounded image cards, subtle stat overlays, animated scroll reveals, smooth carousel transitions, and a refined luxury wellness tone. Avoid harsh colors, heavy shadows, clutter, or generic SaaS styling.
