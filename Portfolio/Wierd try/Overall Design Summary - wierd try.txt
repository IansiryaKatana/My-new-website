Overall Design Summary

This reference is a high-end personal portfolio / creative developer / web designer landing page. The screen communicates a premium, minimal, editorial web presence for a designer or creative studio.

The page feels like a designer portfolio for a Webflow/Framer/creative developer profile, focused on building trust through visual restraint, portfolio proof, and curated expertise.

3. Motion Graphics Breakdown
Overall Motion Direction

Motion should be subtle, premium, and restrained. The design should not feel like a loud animated website. It should feel smooth, elegant, and intentional.

Recommended animation style:

Soft fade
Slight upward movement
Gentle scale
Staggered reveals
Slow parallax on images
Smooth hover transitions
Page Load Animation
Sequence
Header fades in from opacity 0 to 1.
Yellow eyebrow label appears.
Hero heading reveals line-by-line.
Supporting paragraph fades in.
CTA buttons fade/slide upward.
Central portrait card scales from 0.96 to 1.
Side preview cards slide slightly inward behind portrait.

Inferred timings:

Element	Duration	Delay
Header	500ms	0ms
Eyebrow	400ms	150ms
Heading lines	700ms	250ms
Paragraph	500ms	500ms
CTA	450ms	650ms
Portrait card	800ms	750ms
Side cards	700ms	850ms

Easing:

cubic-bezier(0.22, 1, 0.36, 1)
Scroll Reveal Behavior

Each major section should reveal with:

opacity: 0 → 1
translateY: 24px → 0
Optional blur: blur(6px) → blur(0)
Recommended Trigger

Start animation when section enters 15%–25% of viewport.

Hero Image Motion

The central portrait can have very subtle parallax:

Portrait moves upward slower than scroll
Side cards move at a slightly different rate
Max movement should stay below 20px–30px

Do not rotate aggressively. The reference is clean and centered.

Project Mockup Animations

For the blurred website preview cards:

6. Reproduction Notes
Critical Details Not to Miss
1. Hero Must Stay Centered

The reference depends heavily on exact central alignment.

Do not shift the hero into a generic left/right SaaS layout. The heading, CTA, and portrait must stay centered.

2. Portrait Card Is the Main Identity Anchor

The portrait card must:

Sit in the center
Have rounded corners
Use a strong editorial crop
Overlap or visually dominate side mockups
Cast a soft shadow
3. Use Large Negative Space

The layout should breathe. Avoid crowding sections.

This reference uses spacing as a design feature.

4. Maintain White / Black Contrast Rhythm

The page rhythm depends on alternating:

White hero
Light grey stats
White project previews
Black services section
White process/testimonials/articles
Black footer
5. Keep Typography Understated

Avoid overly bold or oversized type. The design feels premium because the typography is calm and precise.

6. Cards Must Feel Soft, Not Heavy

Use:

Light shadows
Subtle borders
Rounded corners
Muted backgrounds

Avoid harsh outlines or bulky UI.

7. Dark Section Must Feel Editorial

The black services section should not look like a generic feature grid. It should feel like a refined content break with sparse layout and careful spacing.

8. Testimonials Must Use Collage Composition

Do not convert testimonials into a normal carousel unless required. The scattered card layout is a key visual feature.
