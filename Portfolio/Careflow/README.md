# CareFlow

CareFlow is a TanStack Start, React 19, TypeScript, Vite, and Tailwind CSS v4 telemedicine dashboard based on the supplied healthcare PRD.

## Stack

- TanStack Start with file-based routing and SSR
- React 19 + TypeScript
- Vite
- Tailwind CSS v4 tokens
- shadcn/Radix-style local UI primitives
- Lucide icons

## Scripts

```bash
npm install
npm run dev
npm run build
```

The dashboard is available at `http://localhost:3000`.
